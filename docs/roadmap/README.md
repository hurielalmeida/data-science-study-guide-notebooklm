# Roadmap de Transição: Engenharia Mecânica para Data Science

## Objetivo

Este roadmap foi elaborado para orientar profissionais da Engenharia Mecânica que desejam migrar para a área de Ciência de Dados.

O plano foi estruturado em seis meses de estudos, combinando fundamentos técnicos, ferramentas de mercado e desenvolvimento de portfólio.

---

## Mês 1 – Fundamentos

### Estudar

* Conceitos de Data Science
* Introdução à Inteligência Artificial
* Estatística Básica
* Probabilidade
* Python para análise de dados
* SQL básico

### Ferramentas

* Python
* Jupyter Notebook
* VS Code

---

## Mês 2 – Manipulação e Preparação de Dados

### Estudar

* Pandas
* NumPy
* Limpeza de Dados
* Tratamento de valores ausentes
* Análise Exploratória de Dados (EDA)

### Projeto

Analisar um conjunto de dados industriais ou de manutenção.

---

## Mês 3 – Machine Learning

### Estudar

* Aprendizado Supervisionado
* Regressão Linear
* Regressão Logística
* Árvores de Decisão

### Ferramentas

* Scikit-Learn

---

## Mês 4 – Avaliação de Modelos

### Estudar

* Train/Test Split
* Cross Validation
* Overfitting
* Underfitting
* Métricas de Avaliação

### Métricas

* MAE
* RMSE
* Accuracy
* Precision
* Recall

---

## Mês 5 – Business Intelligence e Comunicação

### Estudar

* Power BI
* Tableau
* Storytelling com Dados
* LGPD e Ética em IA

### Projeto

Construir dashboards executivos utilizando indicadores industriais.

---

## Mês 6 – Cloud e Portfólio

### Estudar

* Git
* GitHub
* AWS
* Azure
* Fundamentos de Engenharia de Dados

### Projeto Final

Publicar um projeto completo contendo:

* Dataset
* Limpeza dos dados
* Modelagem
* Dashboard
* Documentação

---

## Resultado Esperado

Ao final dos seis meses o profissional deverá possuir:

* Conhecimento em Data Science
* Conhecimento em Machine Learning
* Portfólio no GitHub
* Projetos práticos
* Base para vagas de Analista de Dados Júnior e Cientista de Dados Júnior
