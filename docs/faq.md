# FAQ para Iniciantes em Data Science

## 1. O que é Data Science?

É a área que utiliza dados, estatística, programação e conhecimento de negócio para gerar insights e apoiar decisões.

---

## 2. Qual a diferença entre Data Science e Machine Learning?

Machine Learning é uma subárea da Data Science focada na criação de modelos que aprendem padrões a partir dos dados.

---

## 3. O que é CRISP-DM?

É a metodologia mais utilizada em projetos de dados.

Fases:

- Business Understanding
- Data Understanding
- Data Preparation
- Modeling
- Evaluation
- Deployment

---

## 4. Por que a preparação dos dados é tão importante?

Porque a qualidade dos dados influencia diretamente a qualidade dos resultados obtidos.

---

## 5. Quais linguagens devo aprender?

- Python
- SQL

São as mais utilizadas no mercado de dados.

---

## 6. O que são Features?

Variáveis utilizadas pelo modelo para realizar previsões.

---

## 7. O que são Labels?

Resultados que o modelo tenta prever.

---

## 8. O que é Overfitting?

Quando o modelo decora os dados de treino e apresenta baixo desempenho em dados novos.

---

## 9. O que é Underfitting?

Quando o modelo é simples demais para aprender os padrões relevantes.

---

## 10. O que é validação?

É o processo de testar o modelo em dados que ele nunca viu.

---

## 11. O que é SQL?

Linguagem utilizada para consultar bancos de dados relacionais.

---

## 12. O que é Power BI?

Ferramenta de Business Intelligence utilizada para dashboards e análises.

---

## 13. O que é GitHub?

Plataforma para versionamento, colaboração e portfólio de projetos.

---

## 14. O que é o Kaggle?

Comunidade e plataforma para prática em projetos e competições de dados.

---

## 15. Por que estudar Cloud Computing?

Porque grande parte das soluções modernas de dados roda em nuvem.
