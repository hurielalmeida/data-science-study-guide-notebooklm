# Glossário de Data Science

## Data Science

Área que utiliza dados para gerar conhecimento e apoiar decisões.

## Machine Learning

Técnica que permite que computadores aprendam padrões a partir de dados.

## Feature

Variável de entrada utilizada pelo modelo.

## Label

Resultado que o modelo tenta prever.

## SQL

Linguagem para consulta de bancos de dados.

## CRISP-DM

Metodologia para desenvolvimento de projetos de ciência de dados.

## Overfitting

Quando o modelo memoriza os dados de treino.

## Underfitting

Quando o modelo não consegue aprender os padrões necessários.

## Power BI

Ferramenta de Business Intelligence para criação de dashboards.

## Kaggle

Plataforma para prática e competições de Data Science.
