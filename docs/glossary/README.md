# Glossário de Data Science

## Data Science

Área multidisciplinar que utiliza dados, estatística e programação para gerar conhecimento e apoiar decisões.

---

## Machine Learning

Subárea da Inteligência Artificial que permite que sistemas aprendam padrões a partir dos dados.

---

## CRISP-DM

Metodologia utilizada para desenvolvimento de projetos de Ciência de Dados.

Fases:

1. Business Understanding
2. Data Understanding
3. Data Preparation
4. Modeling
5. Evaluation
6. Deployment

---

## Dataset

Conjunto de dados utilizado para análise ou treinamento de modelos.

---

## Feature

Variável de entrada utilizada pelo modelo para realizar previsões.

---

## Label

Variável alvo que o modelo tenta prever.

---

## SQL

Linguagem utilizada para consulta e manipulação de bancos de dados relacionais.

---

## Python

Principal linguagem de programação utilizada em Ciência de Dados.

---

## Pandas

Biblioteca Python utilizada para manipulação e análise de dados.

---

## EDA (Exploratory Data Analysis)

Processo de exploração dos dados para identificação de padrões, inconsistências e oportunidades.

---

## Overfitting

Quando o modelo aprende excessivamente os dados de treino e perde capacidade de generalização.

---

## Underfitting

Quando o modelo é simples demais para aprender os padrões relevantes dos dados.

---

## Accuracy

Métrica que representa a proporção de previsões corretas realizadas pelo modelo.

---

## Precision

Métrica que mede quantas previsões positivas estavam corretas.

---

## Recall

Métrica que mede quantos casos positivos reais foram encontrados pelo modelo.

---

## Power BI

Ferramenta de Business Intelligence utilizada para construção de dashboards.

---

## Git

Sistema de controle de versão.

---

## GitHub

Plataforma para hospedagem e compartilhamento de projetos utilizando Git.

---

## Cloud Computing

Uso de infraestrutura computacional através da internet.

---

## Kaggle

Plataforma para prática, aprendizado e competições de Ciência de Dados.
